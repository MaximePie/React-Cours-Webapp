Dans cet exercice nous allons créer une application React, pour apprendre à travailler sur de gros projets.

Suivez les instructions de ce lien pour créer votre projet React.
https://create-react-app.dev/docs/getting-started

Effectuez des recherches, et pratiquez sur votre projet pour trouver les réponses aux questions suivantes :

- Qu'est-ce que NPM ? Qu'est-ce que Yarn ? Comment choisir entre les deux ?
- Qu'est-ce que le découpage atomique ?
- Comment importer un composant d'un autre fichier ?
- Qu'est-ce que package.json ? Package-lock ? à quoi servent-ils ?
- Quelles sont les différentes manières de styliser nos composants ?
- Qu'est-ce que npm start ? npm install ? npm i ?
- Comment installer une autre librairie dans notre projet React ?
