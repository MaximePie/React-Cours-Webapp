Créer un gestionnaire de tâches de manière à ce que :

  • L'utilisateur peut saisir une tâche dans un champ texte et l'enregistrer en cliquant sur le bouton "Enregistrer". Utilisez les hooks d'état de React pour les stocker, et stocker le nom de la nouvelle tâche.
  • L'utilisateur peut consulter ses tâches sous forme de liste. Affichez les tâches facilement avec .map.
  • Chaque tâche comporte un bouton "Supprimer" qui permet à l'utilisateur de supprimer la tâche quand il clique dessus.
  • Si la tâche mesure plus de 24 caracètres de long, afficher un message en dessous du champ de texte. Pour le gérer efficacement, vous pouvez utiliser useEffect
